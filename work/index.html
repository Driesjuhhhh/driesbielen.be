Work by Dries Bielen

Explore project case studies, websites, applications, and software work by Dries Bielen.

  • Streetmasters - Website for a car detailing business in Houthalen-Helchteren, Belgium with services, pricing, and contact information.
  • RBGH (Red Bull Gaming Hub) - Volunteer app for the Red Bull Gaming Hub with React, Firebase, Java backend integrations, and operational tooling.
  • OpenInzichten - Community platform for people with invisible illnesses in Flanders with support features and an interactive map.
  • JH Sungarden - Website for a youth center in Zonhoven, Belgium with activities, programs, and community information.
  • MVC Peer - Website for the MVC-Peer volleyball club with VolleyScores API integration for standings and matches.
  • ShiftManager - Shift management system for organizations like youth houses and sports clubs with hour-based point tracking.
  • DRS-Voting - Real-time voting app for live events with audience participation and instant result updates.
  • DRS Stream - Custom remote livestream stack from Raspberry Pi to homelab using Docker-based infrastructure.
  • MosselSystem - Management system for Chiro Wilduraantjes with ordering, kitchen, cash register, and hall management.
  • PingPingDing - Mobile app for checking balance on the school's payment system, built as a course project at PXL.
  • Barbiepoppen Expo - Exhibition website about the history and cultural impact of Barbie dolls, commissioned by PXL University College.
  • Connect 4 - Connect 4 game with a .NET backend, built as a first-year school project.
