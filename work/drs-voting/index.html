About this project

DRS-Voting is a real-time voting application that lets audiences vote for their favorite acts during live events. The system includes secure vote counting, live result updates, and an admin dashboard for organizers.

Key features

  • Real-time voting system
  • Live result updates and visualization
  • Admin dashboard for event management
  • Secure vote counting
  • Support for multiple voting scenarios
  • Cross-platform availability

Challenges

Ensuring reliable real-time synchronization across many concurrent voters while maintaining vote integrity.

What I learned

I gained experience with high-concurrency thinking, live event tooling, and real-time platform design using Appwrite.

DRS-Voting is part of the portfolio of Dries Bielen, a full stack developer from Belgium.
