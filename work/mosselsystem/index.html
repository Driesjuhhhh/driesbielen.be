About this project

MosselSystem is a management application originally built for Chiro Wilduraantjes. It combines order forms, kitchen workflow, cash register operations, and hall management into a single integrated system.

Key features

  • Order form system
  • User login and authentication
  • Kitchen management interface
  • Cash register system
  • Hall management
  • Real-time updates

Challenges

Creating a system that handles multiple connected workflows while remaining user-friendly for non-technical staff.

What I learned

I learned more about practical business applications and building software that supports real operations instead of just demos.

MosselSystem is part of the portfolio of Dries Bielen, a full stack developer from Belgium.
