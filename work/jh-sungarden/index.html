About this project

JH Sungarden is a website for the youth center in Zonhoven that provides information about activities, programs, events, and contact details for local community members.

Key features

  • Activity showcase
  • Program information
  • Contact details
  • Event calendar
  • Responsive design

Challenges

Building a welcoming website for young people that still stays clear and easy to maintain.

What I learned

I learned more about community-focused web design and long-term website maintenance.
